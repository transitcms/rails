Overview

An asset is an uploaded file. Assets can be `standalone` or application-wide resources, or they may be attached directly to a deliverable

def urls
  return { :original => file.url(:original) } unless image?
  file.styles.keys.inject({}) do |hash, style|
    hash.merge!(style => file.url(style))
  end
end
