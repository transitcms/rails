Module: Transit::Extension::Translations

Overview

Adds functionality for translating attributes within models. By stubbing these methods by default, external definitions or extensions can call them regardless of whether translation is enabled or not.

def enable_translation
  begin
    include Mongoid::Globalize        
  rescue LoadError
    raise "Missing the mongoid_globalize gem. Please add it to your gemfile to translate models"
  end
end
